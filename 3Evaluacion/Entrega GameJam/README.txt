Efectos:
	Efecto Parallax:
		Creado siguiendo la guia de, BravePixelG. Aun que no se usa casi nada, también me inspiré en el Parallax de Dani
		El material que le aplica al fondo (las diferentes capas) es para poder usar el Offset del material, por defecto bloquea esta propiedad.
		Como los Sprites pueden tener particulas o efectos (se pueden ver partes que se cortan), se activa la propiedad sliced en el sprite renderer
		para eliminar cualquier efecto que le pueda afectar.
			https://www.youtube.com/watch?v=7bJT6rf-Jvk   (BravePixelG)
			https://www.youtube.com/watch?v=zit45k6CUMk   (Dani, está en ingles)

	Seguimiento del personaje por parte de la cámara:
		Script creado con ayuda de IA a partir de especificaciones propias. Basandome en el script del Pong.
